# M5 上线与提审检查清单

本清单对应开发文档 M5。代码侧已包含隐私授权入口、免责声明、空态/加载态/骨架屏、提交频控、管理员登录锁定、结果归属校验和生产配置约束。带“平台操作”的项目需在微信公众平台或云服务器上完成。

## 1. 微信公众平台（平台操作）

- [x] 主体类型确认为企业。
- [ ] 建议主类目选择“工具 → 信息查询”（以微信公众平台当前实际显示名称为准）。本程序仅提供问卷作答、规则计分和报告查询，不提供课程、职业培训、医疗诊断或心理治疗。
- [ ] 不选择“教育 → 职业培训”，除非企业确实开展培训且能提交平台要求的办学资质；不选择医疗/心理类目。
- [ ] 在“设置 → 服务内容声明 → 用户隐私保护指引”如实声明：微信身份标识、用户主动填写的昵称、用户主动选择的头像、测评作答、测评结果。
- [ ] 声明“选择照片或拍摄”，用途填写“用于用户主动选择或拍摄个人头像并展示在测评报告中”。
- [ ] 在隐私保护指引中声明“保存图片到相册”，用途填写“用于用户保存职业锚报告长图，以及管理员保存邀请小程序码”。
- [ ] 在隐私保护指引中声明“写入剪贴板”，用途填写“用于管理员复制邀请码并进行分发”。
- [ ] 隐私保护指引审核生效后，在开发版真机上分别验证“同意”“不同意”“查看指引”三条路径。
- [ ] 将 `https://api.<域名>` 配置为 request 合法域名。
- [ ] 上传体验版，确认基础库不低于 2.20.1，完成全部真机回归后提交审核。
- [ ] 提审说明明确：本产品用于职业兴趣与价值观探索，结果仅供参考，不提供医学、心理诊断或职业决策结论。
- [ ] 按 `M5企业主体提审材料.md` 准备审核说明和可多次使用的审核邀请码。

## 2. 域名与服务器（平台操作）

- [ ] 域名已完成 ICP 备案，HTTPS 证书链完整且在有效期内。
- [ ] 使用 `career-anchor-server/deploy/nginx-career-anchor.conf.example` 配置 TLS 1.2/1.3、单 IP 10 请求/秒限流，仅将 Spring Boot 暴露给本机 Nginx。
- [ ] 防火墙公网只开放必要端口；MySQL 3306 和 Spring Boot 8080 不对公网开放。
- [ ] 配置日志轮转、磁盘空间和证书到期提醒。

## 3. 生产环境变量

`prod` profile 启动时必须提供以下变量，不得写入 Git：

```text
DB_URL
DB_USERNAME
DB_PASSWORD
JWT_SECRET
WECHAT_APP_ID
WECHAT_APP_SECRET
```

- [ ] `JWT_SECRET` 使用至少 32 字节的随机值，且不复用数据库或管理员密码。
- [ ] 管理员密码以 BCrypt 哈希存入 `admin_accounts`；生产环境不设置 `ADMIN_INITIAL_PASSWORD`。
- [ ] 小程序端 `baseUrl` 改为 `https://api.<域名>/api`，无 IP、localhost 或 HTTP 地址。

## 4. 发布前回归

- [ ] 首次进入会展示隐私提示；拒绝后不登录，同意后可继续；再次进入不重复弹出本地提示。
- [ ] 用户填写昵称并选择头像后，管理端当前及历史测评结果均显示该昵称；资料未完善时不能开始新测评。
- [ ] 真机验证结果长图和邀请小程序码可保存到相册，管理员邀请码可复制到剪贴板，不出现 `api scope is not declared in privacy agreement`。
- [ ] 首页和报告页始终显示“仅供参考，不构成任何诊断或决策依据”。
- [ ] 历史报告、报告详情、题目加载、管理概览与列表均有加载状态；无数据有空态；失败可重试。
- [ ] 同一用户 60 秒内第二次提交返回 429 / `42002`，失败提交不会错误占用频控窗口。
- [ ] 普通用户不能查看他人报告或调用管理员接口；管理员连续输错 5 次锁定 10 分钟。
- [ ] 邀请码过期、用尽、停用及并发消耗行为正确；日志中不出现完整 openid、邀请码、Token 或密钥。
- [ ] 执行后端全部测试与打包、小程序全部测试、JS/JSON 语法检查，并在微信开发者工具执行代码质量检查和真机预览。
