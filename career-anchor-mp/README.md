# Career Anchor Mini Program

M5 微信原生小程序，包含完整用户测评体验和小程序内管理控制台，并补齐首次隐私授权、免责声明、空态、错误态与骨架加载体验。管理员可二次鉴权，查看统计与全部结果，批量生成/停用邀请码，生成、保存和转发邀请小程序码；扫码 `scene` 复用普通邀请码校验与消耗链路，免输码直达测评。

## 微信云托管联调

1. 在微信云托管中部署后端并发布版本。
2. 在微信开发者工具中导入本目录。
3. 在 `utils/config.js` 中配置微信云托管环境 ID 和服务名称。小程序通过 `wx.cloud.callContainer` 调用服务，无需配置默认公网域名。
4. 将真实 appid 写入本地 `project.private.config.json`（该文件已忽略），不要提交密钥。

二维码入口页面固定为 `pages/quiz/index`，scene 格式为 `i=AB3K9XQ2`。可在开发者工具的编译模式中添加 scene 参数验证扫码流程。

`dev` 后端首次启动会在管理员表为空时创建默认密码 `admin123`；该密码仅供本地联调，可通过后端环境变量 `ADMIN_INITIAL_PASSWORD` 覆盖。生产环境不会自动创建管理员。

首次进入会先展示数据使用说明，并通过 `wx.requirePrivacyAuthorize` 衔接微信《用户隐私保护指引》。正式体验或提审前，必须先在微信公众平台完成隐私保护指引配置；平台与服务器侧操作见 `../docs/M5上线检查清单.md`。
